# 员工 30 秒接入 cockpit 机器人 — 全员图文手册

## 适合谁
公司任何一位员工：设计师 / 工厂 / 主播 / 客服 / 老板娘家人 / 不懂技术的同事

## 你需要的设备
✅ 一部手机
✅ 安装飞书 app（公司账号已登录）
**没了。**

---

## 第 1 步：找到 cockpit 机器人

打开飞书 app → 顶部搜索框 → 输入「lark-fashion-cockpit」

```
┌──────────────────────────┐
│ 🔍 lark-fashion-cockpit  │ ← 搜
└──────────────────────────┘

搜索结果：
┌──────────────────────────────┐

```

## 第 2 步：发起会话

点机器人头像 → 看到一个简介页 → 点「发起会话」按钮

```
┌──────────────────────────────┐
│  🤖                           │
│  lark-fashion-cockpit 助手   │
│  ──────────────────          │
│  女装运营驾驶舱 AI 助手       │
│                              │
│  [⭐ 关注] [💬 发起会话]      │ ← 点这个
└──────────────────────────────┘
```

## 第 3 步：直接说人话

进了私聊窗口（**只有你和机器人能看到**），直接说想做啥：

```
你: 你能帮我做什么

机器人: 👋 欢迎！你是 [设计师 · 马萍蔓] 角色，可以用：
   📋 看下我的任务  →  task-tracker
   🛍 DRS-XXX 配什么  →  product-matching
   📚 查产品元素标签  →  product-library
   🎨 文档迭代  →  doc-iterator
   📖 查知识库 / 词典  →  knowledge-base
   ...

你: 我这周交了几张稿
机器人: [私下回复你专属的卡片]
   你本周已交付 12 张稿
   平均交付提前 14m
   3 张被改稿
   ...
```

---

## 私聊和群聊有啥区别

| 场景 | 群聊 | 私聊 |
|---|---|---|
| 谁看得到对话 | 群里所有人 | **只有你和机器人** |
| 适合 | 团队公开讨论 | **个人查询、敏感数据** |
| 老板娘看得到吗 | 看得到群聊内容 | **看不到你私聊内容**，但能看你触发了什么 skill（审计用）|

## 你能干的几件事（按角色不同）

### 🎨 设计师（马萍蔓）
- "我这周交了几张稿"
- "DRS-0429 这款想配个外套"
- "灵感板里有啥新东西"

### 🏭 工厂主管（申丽媛）
- "今天的紧急生产任务"
- "DRS-0429 库存还有几件"
- "这批面料什么时候到"

### 📺 主播
- "晚上直播带什么款"
- "这款库存够卖一场吗"
- "上场直播 GMV 多少"

### 💬 客服
- "客户赵姐之前买过啥"
- "DRS-XXX 退货啥情况"
- "查 [客户名] 的咨询历史"

### 👑 老板娘（全部权限）
- 任何 cockpit 能力都可以用

---

## 隐私保证

- ✅ 你的私聊内容飞书原生加密，连飞书官方都看不到
- ✅ 老板娘也看不到你的对话内容（但能看你"用了什么 skill"做审计）
- ✅ 群聊和私聊数据物理隔离

- 试着说人话，机器人听不懂会教你
- 看公司 wiki 的「cockpit 使用手册」
