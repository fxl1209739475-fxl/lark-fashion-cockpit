## 📱 拍摄技术建议

### 工具
- **直接用飞书自己录屏**（手机飞书有内置录制）
- 或者 **OBS Studio**（PC 免费）
- 配音：**飞书妙记 AI 配音** 或者你自己声音（抖音博主背景，自己出镜效果最好）

### 节奏
- **每 3 秒切一个画面**，不要长镜头
- **关键数据闪到屏幕中央 + 红圈圈** 老板娘看得到的位置
- **手机录屏 + 你自己出镜小窗** 在右下角（建立"真人老板娘"信任感）

### 关键画面清单
1. cockpit 工作台 widget 首页（已有 demo HTML 截图）
2. 飞书私聊"今天卖得咋样" → 卡片回复（已有截图）
3. 马萍蔓任务完成截图（memory 里 14 分 41 秒）
4. 私聊"以马萍蔓身份" → 红色权限拒绝（你今晚刚截的）
5. 视频脚本拆解结果（你测过的"姐姐"那条）
6. 老板分身 200 题语料表
7. 自动调度器 12 任务清单
8. GitHub README 顶部
