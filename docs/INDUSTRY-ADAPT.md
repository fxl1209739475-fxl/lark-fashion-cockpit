## 🔄 业务规则适配层（需要改写）

### 适配核心要点

每个行业需要重新定义 4 个东西：
1. **元素标签体系**（5 维 32 选项）
2. **互补品类规则**（搭配逻辑）
3. **N 步流程模板**（女装 11 步上新 → 各行业不同）
4. **角色映射**（女装 9 角色 → 各行业不同）

---

### 🧴 化妆品 / 美妆行业适配

#### 元素标签体系（5 维）

| 维度 | 选项 |
|---|---|
| **品类** | 底妆 / 眼妆 / 唇妆 / 香水 / 护肤 / 工具 |
| **功效** | 美白 / 抗老 / 保湿 / 控油 / 修复 / 防晒 |
| **肤质** | 干皮 / 油皮 / 混油 / 敏感 / 中性 |
| **场景** | 日妆 / 夜妆 / 通勤 / 派对 / 婚礼 / 旅行 |
| **风格** | 韩系 / 日系 / 欧美 / 中国风 / 极简 |

#### 互补品类规则

```
底妆 ↔ 遮瑕 / 定妆 / 散粉
眼妆 ↔ 眉毛 / 睫毛 / 眼影
唇妆 ↔ 唇釉 / 口红 / 唇线笔
护肤 ↔ 化妆水 / 精华 / 面霜（黄金组合）
香水 ↔ 单独售卖（无强搭配）
```

#### 替换 launch-decision 4 维信号

| 原（女装）| 新（化妆品）|
|---|---|
| 元素相似爆款 | 同肤质 + 同功效爆款 |
| 滞销避雷 | 同产品类型滞销品 |
| 退货风险 | 过敏 / 色差退货风险 |
| 人为操作教训 | 配方调整 / 包装变更教训 |

#### 上新 N 步流程

```
原女装 11 步：选款→设计稿→打版→大货→打样→拍摄→主图→详情页→内容→直播→上架→投放
化妆品 9 步：研发立项→配方测试→送检→注册备案→大货→拍摄→详情页→内容→上架投放
```

#### 角色映射

| 女装 | 化妆品 |
|---|---|
| 设计师 | 配方师 |
| 打版师 | 检测员（备案/SGS）|
| 生产主管 | 生产主管 |
| 模特 | 美妆达人 |
| 摄影师 | 摄影师 |
| 主播 | 主播 |
| 内容编辑 | 内容编辑 |
| 客服 | 客服（含敏感咨询）|
| 运营 | 运营 |

**适配工作量：1-2 天改字段 + 业务规则**

